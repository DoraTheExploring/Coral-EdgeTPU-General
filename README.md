## Docker
### Installing Docker

```console
wget https://download.docker.com/linux/static/stable/aarch64/docker-19.03.8.tgz
tar xvf docker-19.03.8.tgz
sudo cp docker* /usr/bin/
```

Optionally go here: https://download.docker.com/linux/static/stable/aarch64/
